# AI Voice Client

自用WebSocket 客户端应用，支持连接管理、断线重连、心跳保活、消息格式化等功能。
测试语音传输

## 功能特性

- ✅ WebSocket 连接管理
- ✅ 自动断线重连（指数退避）
- ✅ 心跳保活机制
- ✅ 长时间无消息自动断开
- ✅ 自定义消息格式化
- ✅ 文件传输支持
- ✅ Web 界面管理

## 快速开始

### 安装依赖

```bash
npm install
```

### 运行应用

```bash
npm start
```

### 开发模式

```bash
npm run dev
```

### 运行测试

```bash
npm test
```
